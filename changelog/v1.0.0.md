# Changelog

## v1.0.0 (2026-06-01)

Initial release of KCraft - a Minecraft launcher built with Rust and Slint.

### Features

Minecraft instance management with full launch pipeline. Microsoft MSA authentication (device code flow). Offline account support. Ely.by / Authlib-Injector authentication. Mod platform integration: CurseForge, Modrinth, FTB, ATLauncher, Technic, Packwiz. DAG-based dependency resolver for modpack building. Async download engine with resume, validation, and progress. Java runtime auto-detection. Native Slint GUI (no webview). Cross-platform: Linux, Windows, macOS. Dark/light theme toggle. Instance CRUD and settings editor. Modpack builder with JAR parsing and conflict detection. Crash reporter.

### Bug Fixes

Fixed data_root directory name from kcrack to kcraft. Fixed LD_LIBRARY_PATH override to append instead of replace. Fixed production log level from debug to info. Fixed progress bar width formula for Slint float-to-length conversion. Fixed modpack builder message from Installed X packages to Added X file(s).

### Maintenance

100% Rust codebase. Tauri+Nuxt frontend replaced with Slint native GUI. Modular crate structure with single-responsibility modules. CI: lint, format, test on Linux/Windows/macOS. Automated release builds with GitHub Actions.
